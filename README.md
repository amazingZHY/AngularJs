#### * 关于AngularJS指令及过滤器  
    E - 指令可以作为HTML元素使用
    A - 指令可以作为HTML属性使用
    C - 指令可以作为CSS类使用
    M - 指令可以在HTML注释中使用 
过滤器可以使用一个管道字符（|）添加到表达式和指令中 。
